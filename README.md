# Sistema Universidade

Projeto desenvolvido em Java com Spring Boot para simular o ambiente acadêmico de uma universidade.

## Funcionalidades

### Alunos

* Cadastro de alunos
* Consulta de alunos pela matrícula
* Armazenamento de nome, endereço, matrícula e data de ingresso
* Exibição das matérias matriculadas e da média do aluno

### Matérias

* Cadastro de matérias
* Consulta de matérias pelo código
* Registro de código e ementa da disciplina

### Turmas

* Cadastro de turmas
* Consulta de turmas pelo código
* Exibição dos alunos matriculados
* Registro de três notas por aluno
* Verificação automática de aprovação (média maior ou igual a 6)

## Tecnologias Utilizadas

* Java
* Spring Boot
* MySQL
* HTML
* CSS
* JavaScript

## Objetivo

Desenvolver um sistema acadêmico que atenda aos requisitos propostos na disciplina, aplicando conceitos de Programação Orientada a Objetos, APIs REST e persistência de dados.
